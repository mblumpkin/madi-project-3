# Beach Sand Art — Interactive Web Experience

## Concept

Beach Sand Art is a calm, interactive digital beach environment designed to create a relaxing, zen-like experience. Users can draw with falling sand while ambient ocean sounds and animated environmental elements create an immersive mood. The piece focuses on atmosphere, motion, and playful interaction to simulate a peaceful moment at the beach.

The experience emphasizes calm exploration, allowing users to create temporary sand art while the world subtly animates around them.

---

## Interactions

The project includes multiple interactive behaviors:

* **Click + Drag** — Users create falling sand particles that stack and flow naturally
* **Color Picker** — Change the sand color to create different artwork
* **Flow Slider** — Control how much sand is released while drawing
* **Clear Button** — Reset the canvas and start over
* **Enter Button** — Starts the experience and enables ocean audio (user-initiated)

These interactions allow users to experiment and remix the scene.

---

## Motion & Animation

Animation is a primary part of the experience. The environment continuously moves to create a living scene:

* Animated day-to-night sky gradient
* Moving sun with glow effect
* Water shimmer and animated waves
* Flying birds across the sky
* Walking crab along the beach
* Falling sand particle physics simulation

All animation is created using JavaScript canvas rendering and `requestAnimationFrame`.

---

## Media & Atmosphere

* Ocean wave audio (user-initiated)
* Animated beach environment
* Soft UI overlay with glassmorphism styling

The goal is to create a relaxing, ambient digital artwork.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Canvas API)
* requestAnimationFrame animation loop

No external libraries were used.

---

## External Assets

* Ocean sound effect (`audio/oceansounds.mp3`)
* Crab image (`img/crab.png`)
* Favicon (`img/wave.webp`)

All assets are used for visual atmosphere within the interactive scene.

---

## Responsive Design

The experience resizes dynamically to fit different screen sizes. Canvas elements automatically scale to the viewport, allowing the interaction to work on both desktop and mobile devices.
